# vim-gluon

## Description

This is a vim plugin that provides [gluon][g] syntax highlighting (more to come)

## Installation

### Using [Pathogen][p]

```shell
git clone --depth=1 https://github.com/salpalvv/vim-gluon.git ~/.vim/bundle/rust.vim
```

## Features

Just syntax highlighting for now
